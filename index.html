主要项目 2025.12 - 2026.05

基于 Spring Boot 的前后端分离外卖平台，包含用户端、商家端、管理端三端，涵盖菜品管理、购物车、订单全流程、用户认证、报表统计等核心功能。

技术栈
Spring Boot MySQL Redis JWT Nginx WebSocket
性能优化

使用 Redis 缓存菜品分类数据，结合 @Cacheable/@CacheEvict 注解管理缓存生命周期，减少高频查询数据库压力

实现管理端 + 商家端完整业务模块，包括员工管理、菜品/套餐 CRUD、分类管理、订单状态机流转（待接单→备餐→派送→完成/取消）

使用 JWT + 自定义拦截器实现双端（管理员/用户）Token 鉴权，做到接口级权限隔离

集成阿里云 OSS 实现菜品图片上传，封装统一上传工具类，支持文件类型校验与 URL 回显

采用 WebSocket 推送新订单提醒与催单通知，实现商家端实时消息感知

利用 Apache POI 导出近 30 日营业额、订单量、客单价等运营数据 Excel 报表
