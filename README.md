# 罗浩宇 - 个人简历网站

🔗 在线访问：https://justin1uo.github.io/portfolio

## 📋 内容

- `index.html` - 单页简历网站
- `resume.docx` - 可下载简历

## 🚀 部署到 GitHub Pages

### 步骤 1：创建 GitHub 仓库

1. 登录 GitHub：https://github.com
2. 点击右上角 **+** → **New repository**
3. 仓库名称填写：`portfolio`
4. 选择 **Public**
5. 点击 **Create repository**

### 步骤 2：上传文件

在仓库页面，点击 **uploading an existing file**，将以下文件拖入：
- `index.html`
- `resume.docx`

### 步骤 3：启用 GitHub Pages

1. 进入仓库 **Settings**（设置）
2. 左侧菜单找到 **Pages**
3. **Source** 选择 **Deploy from a branch**
4. **Branch** 选择 `main`，文件夹选择 `/ (root)`
5. 点击 **Save**

### 步骤 4：等待部署

等待 1-2 分钟，刷新页面，你的网站就会出现在：
```
https://your-username.github.io/portfolio
```

## 🛠️ 本地预览

直接用浏览器打开 `index.html` 即可预览。

## 📝 自定义

如需修改内容，直接编辑 `index.html` 文件即可。
